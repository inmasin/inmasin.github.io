Case Study

How to measure grain size

One method for getting accurate grain-size measurements is sieve analysis using ASTM-standard-size sieves (sometimes, other classification systems with different standards for sieve sizes are used). In this method, soil particle size is measured as soil passes through sieves to capture the different size ranges.
